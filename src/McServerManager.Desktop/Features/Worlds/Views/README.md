# World Views

## Purpose
Contains the WinUI window markup and minimal code-behind for the worlds feature.

## Contents
- `WorldsWindow.xaml`: split-pane shell layout.
- `WorldsWindow.xaml.cs`: startup and selection event forwarding.

## Dependency Rules
- Keep code-behind limited to view glue and event forwarding.

## Change Notes
- Update when the shell adds new panes or dialog entry points.
