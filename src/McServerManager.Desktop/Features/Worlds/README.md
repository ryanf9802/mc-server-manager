# Worlds Feature

## Purpose
Implements the main workflow for CRUD operations on managed worlds and explicit activation of live files.

## Contents
- `Views/`: WinUI windows and XAML.
- `ViewModels/`: stateful presentation logic.

## Dependency Rules
- Keep raw SFTP and validation logic out of this feature; consume application services instead.

## Change Notes
- Update this README when world CRUD flow or shell layout changes.
