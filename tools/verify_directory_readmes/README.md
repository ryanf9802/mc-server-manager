# Verify Directory READMEs

## Purpose

Checks that each committed directory under `docs`, `src`, `tests`, and `tools` contains a local `README.md`.

## Contents

- `check_readmes.py`: repo tree scanner.

## Dependency Rules

- Keep this script self-contained so it can run without project dependencies.

## Change Notes

- Update ignore rules here if new generated directories need to be skipped.
